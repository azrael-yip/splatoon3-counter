# 🐟 Splatoon 3 打工武器紀錄統計工具 (使用說明)

本工具用於讀取 **stat.ink** 的打工 (Salmon Run) 紀錄，並統計您在特定班表內使用過的武器次數。

為了使用此工具，您需要先將 Nintendo Switch 帳號綁定至第三方工具（如 **KOOK** 上的機器人「小魷魷」），將資料同步至 stat.ink，最後匯出 JSON 檔案。

---

## 📋 前置準備

- [x] 擁有 **Nintendo Switch Online** 帳號。
- [x] 擁有 **[stat.ink](https://stat.ink/)** 帳號。
- [x] 擁有 **KOOK** 帳號（用於使用「小魷魷」服務）。

---

## 🚀 步驟一：綁定 NS 帳號至「小魷魷」

「小魷魷」是一個方便的第三方工具，可以協助您將遊戲資料同步上傳。

1.  在 KOOK 中找到「小魷魷」並點擊首頁的 **「登錄 / 綁定」**。
2.  按照指引登入您的 **Nintendo Account**。
3.  綁定成功後，您應該能在小魷魷內看到您最近的對戰與打工紀錄。

## 🔄 步驟二：設定 stat.ink 同步

1.  登入 [stat.ink](https://stat.ink/) 網站。
2.  點擊右上角頭像，選擇 **「Profile and Settings (設置)」**。
3.  點擊 **「API Token」** 分頁。
4.  複製您的 `API Token`。
5.  回到「小魷魷」：
    * 按指引進行同步設定。
    * 貼上 API Token。
6.  **請等待同步完成**，確保您的打工紀錄已成功上傳至 stat.ink。

## 📥 步驟三：下載打工資料 (JSON)

1.  前往 stat.ink 的個人資料匯出頁面：[https://stat.ink/profile](https://stat.ink/profile)
2.  向下滑動找到 **「匯出 (斯普拉遁3)」** 區域。
3.  找到 **Salmon Run (JSON)** 選項。
4.  通常檔名格式為：`statink-salmon-[您的ID].json.gz`。
5.  點擊下載該檔案。

## 📊 步驟四：使用本統計工具

1.  **手動解壓縮檔案**：
    > ⚠️ **注意**：本工具不支援直接讀取 `.gz` 壓縮檔。請使用解壓縮軟體（如 WinRAR, 7-Zip）將下載的檔案解壓，取得 `.json` 檔案。
2.  用瀏覽器開啟本工具的 `index.html` 檔案。
3.  **直接拖曳** 解壓縮後的 `.json` 檔案到網頁中間的 **藍色虛線框** 內。
4.  工具會自動執行以下動作：
    * 解析資料。
    * 自動鎖定 **最新一場打工** 的時間與場地。
    * 過濾掉非本次班表的舊紀錄。
    * 統計並顯示該梯次您使用過的武器次數。

## 💡 常見問題 (Q&A)

**Q: 為什麼顯示「沒有新紀錄」？**
> **A:** 請確認您在「小魷魷」是否已經執行過同步，且 stat.ink 網站上已經能看到最新的打工紀錄。此外，本工具預設只計算「最新一輪班表」，如果是舊班表的資料會被自動過濾。

**Q: 下載的檔案是 `.gz` 結尾，需要解壓縮嗎？**
> **A:** **需要**。本工具不內建解壓縮功能，需要您先手動解壓成 `.json` 格式後再拖入網頁。

**Q: 為什麼有些武器顯示為「未知」？**
> **A:** 官方若推出了新賽季武器，stat.ink 的代碼可能會更新，導致本地字典對應不上。

**Q: 我想看歷史所有場次的統計，不只是這一輪？**
> **A:** 目前工具設計為「鎖定最新班表」，以提供最具參考價值的輪替數據。
